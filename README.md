### Figma To Tailwind

- Frame = 宽度 高度 圆角
- Layout = 间距 方向 边距 对齐
- Fill = 填充色
- Stroke = 描边
- Effect = 效果

- 高度
  - Fixed: h-[px]
  - Hug: h-fit
  - Fill: h-full / h-max
- 宽度

  - Fixed: w-[px]
  - Hug: w-fit
  - Fill: w-full / w-max

- 文字
  - 字符
    - 字体
    - 字重
    - 字号
  - 样式
    - 斜体
    - 下划线
    - 删除线
  - 排版
    - 分布
    - 大小写
  - 间距
    - 字间距
    - 词间距
    - 行间距
